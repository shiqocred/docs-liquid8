---
title: "Create"
---

- Buka halaman “Migrate -> Destination”, lalu klik button Add Destination. Akan muncul tampilan create destination. Inputkan data sesuai dengan data toko yang akan menjadi tujuan. **Nama** untuk memberikan nama toko, **No. Phone** untuk memberikan nomer telephone, **Address** untuk menentukan alamat toko (bisa menggunakan pin map pada bagian kiri form).
  ![ Create Destination](/image/outbound/migrate-color/crud-destination/create/a-1.png " Create Destination")
  ![ Create Destination](/image/outbound/migrate-color/crud-destination/create/a-2.png "Create Destination")

- Jika sudah sesuai klik button Create, alur **CRUD Create Destination** selesai.
