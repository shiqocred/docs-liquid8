Berikut Gambar beserta Flow penggunaan halaman <BU>Manual Inbound</BU>:

## Flow

<Steps>
  <Step>
    Buka halaman <BU>Manual Inbound</BU>
  </Step>

  <Step>
    Isi semua kolom data
  </Step>
  
  <Step>
    Jika harga Rp. 100.000 atau diatasnya maka pilih kategori terlebih dahulu, atau jika barang mengalami kerusakan maka isi deskripsi <BU>Damaged</BU> atau <BU>Abnormal</BU> terlebih dahulu
  </Step>

  <Step>
    Klik <BU>Submit</BU> untuk menyelesaikan
  </Step>
</Steps>

## Tabel Deskripsi

| Kolom                 | Deskripsi                                            | Keterangan                               |
|-----------------------|------------------------------------------------------|------------------------------------------|
| Product Name          | Nama Barang                                          | ✅ Edited, Required                      |
| Price                 | Harga Barang                                         | ✅ Edited, Required                      |
| Qty                   | Kuantitas Barang                                     | ✅ Edited, Required, Angka, Min. 1       |
| Price After Discount  | Harga setelah diskon (kategori) atau harga fix color | ✅ Edited, Required                      |
| Good: Kategory        | Pilih salah satu                                     | ✅ Edited, harga Rp. 100.000 / diatasnya |
| Good: Color           | Color                                                | ❌ Not Edited, dibawah harga Rp. 100.000 |
| Abnormal: Description | Deskripsi barang Abnormal                            | ✅ Edited, Required                      |
| Damaged: Description  | Deskripsi barang Damaged                             | ✅ Edited, Required                      |
