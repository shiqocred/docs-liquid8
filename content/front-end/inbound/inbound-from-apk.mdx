Berikut Gambar beserta Flow penggunaan halaman **Inbound Form Apk**:

## Flow

<Steps>
  <Step>
    Buka halaman **Product Input**.
  </Step>

  <Step>
    Pilih salah satu [<BU>Action</BU>](#tampilan-awal-1) berikut: `Add to filter`, `Detail` dan `Delete`
  </Step>
  
  <Step>
    Buka `list filter barang` dengan menekan tombol **Filtered Products**.
  </Step>

  <Step>
    Tekan [<BU>Action</BU>](#tampilan-filtered-2) `Delete` untuk menghapus atau Klik **Done Check All** untuk memindahkan barang di `list filter` ke `stagging`.
  </Step>
</Steps>

## Tabel Action

### Tampilan Awal

| Action        | Icon            | Deskripsi                                                        |
|---------------|-----------------|------------------------------------------------------------------|
| Add to filter | <PlusCircle />  | Menambah barang ke `list filter` untuk dipindahkan ke `stagging` |
| Detail        | <ReceiptText /> | Membuka dan memperbarui data barang                              |
| Delete        | <Trash2 />      | Menghapus data barang secara permanen                            |

### Tampilan Filtered

| Action        | Icon       | Deskripsi                                |
|---------------|------------|------------------------------------------|
| Delete        | <Trash2 /> | Menghapus data barang dari `list filter` |
