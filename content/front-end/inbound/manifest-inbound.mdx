## Kelola Data

import { ArrowUpRightFromSquare, ShieldCheck, ReceiptText, Trash2 } from "lucide-react"

File yang berhasil di unggah akan disimpan di halaman <BU>Manifest Inbound</BU>
        
<ImageZoom src="/image/inbound/manifest/c-1.png" width={3276} height={2018} alt="Upload Inbound Process" className="rounded-lg m-0" />

### Deskripsi header tabel

| Nama          | Deskripsi                                          | Ket                                                               |
|---------------|----------------------------------------------------|-------------------------------------------------------------------|
| Document Name | Nama dari dokumen yang di unggah di Inbound Proses | -                                                                 |
| Date          | Tanggal file di unggah                             | -                                                                 |
| Total Items   | Total barang di dalam data                         | -                                                                 |
| Status        | Status Data                                        | [<ArrowUpRightFromSquare className='size-4'/>](#list-status-data) |
| Action        | Opsi untuk data tersebut                           | [<ArrowUpRightFromSquare className='size-4'/>](#list-action-data) |

### List status Data

| Nama          | Deskripsi                             |
|---------------|---------------------------------------|
| In Progress   | Data dalam proses penginputan barang  |
| Pending       | Data belum diproses atau data gagal   |
| Done          | Data telah selesai di proses          |

### List action Data

| Nama   | Icon                               | Deskripsi                             |
|--------|------------------------------------|---------------------------------------|
| Check  | <ShieldCheck className='size-4' /> | Melakukan input barang sesuai data     |
| Detail | <ReceiptText className='size-4' /> | Menampilkan halaman detail data       |
| Delete | <Trash2 className='size-4' />      | menghapus data                        |

- Pertama upload data excel dengan cara buka pada halaman “Inbound Process”, setelah terupload lalu dipilih header yang diperlukan.
  ![Inbound Process](/image/inbound/manifest/a.png "Inbound Process")
- Lalu jika sudah terupload data akan masuk ke dalam halaman “Manifest Inbound”, lalu klik detail pada data yang akan di cek.
  ![Detail Manifest Inbound](/image/inbound/manifest/b.png "Detail Manifest Inbound")
- Sebelum memulai pengecheck an barang, “jika” ingin menggunakan custom barcode maka klik barcode di pojok kanan atas dan beri format depan pada barcode. Jika sudah klik next di atas list barang.
  ![Detail Manifest Inbound](/image/inbound/manifest/c.png "Detail Manifest Inbound")
- Setelah klik next akan menuju ke halaman “Check”, selanjutnya masukkan barcode barang, pilih kategori “jika” barang diatas 100k, “jika” di bawah 100k tidak pilih kategori, selanjutnya pilih qualitas (abnormal, damaged, lolos)
  ![Detail Manifest Inbound](/image/inbound/manifest/d.png "Detail Manifest Inbound")
  ![Detail Manifest Inbound](/image/inbound/manifest/e.png "Detail Manifest Inbound")
  ![Detail Manifest Inbound](/image/inbound/manifest/f.png "Detail Manifest Inbound")
- Terakhir klik submit jika sudah dipilih kategori/quality product nya. **Manifest Inbound** selesai.
