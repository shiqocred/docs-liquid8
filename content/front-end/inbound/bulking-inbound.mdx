Berikut Gambar beserta Flow penggunaan halaman <BU>Bulking Inbound</BU>:

## Flow

<Steps>
  <Step>
    Buka halaman <BU>Bulking Inbound</BU>
  </Step>

  <Step>
    Upload File dengan cara _Drag and Drop_ atau _Klik Area_
  </Step>
  
  <Step>
    Pilih <BU>Tipe Bulking</BU> Product:
    - Category
    - Color
  </Step>

  <Step>
    Klik <BU>Save</BU> untuk menyelesaikan
  </Step>
</Steps>

<Callout>
  Jika ingin mengganti file dapat menekan tombol <BU>Change File</BU>
</Callout>
