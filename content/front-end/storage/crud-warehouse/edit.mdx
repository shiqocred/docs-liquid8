---
title: "Edit"
---

- Masuk kehalaman “Palet -> Warehouse” lalu klik button Edit (icon pensil). Akan muncul pop up dengan inputan **Name** untuk memberikan nama warehouse, **No. Phone** untuk memberikan kontak terhadap warehouse, **Address** untuk memberikan nama alamat detail warehouse, **Kecamatan** untuk kecamatan, **Kabupaten** untuk kabupaten, **Provinsi** untuk provinsi, **Latitude Longitude** untuk garis lintang dan bujur yang sudah terisi. Untuk **Address** sampai **Longitude** bisa memakai pin map.  
  ![CRUD Edit Warehouse](/image/storage/crud-warehouse/edit/a-1.png "CRUD Edit Warehouse")
  ![CRUD Edit Warehouse](/image/storage/crud-warehouse/edit/a-2.png "CRUD Edit Warehouse")
- Jika sudah klik button **Update.** Alur **Edit Warehouse** selesai.
