- Buka halaman “Palet -> Condition”, lalu klik button Add Condition. Akan muncup pop up dengan inputan **Name** untuk memasukkan nama kondisi, **Slug** untuk membuat name versi uniq (tidak wajib diisi).  
  ![CRUD Condition Create](/image/storage/crud-condition/create/a-1.png "CRUD Condition Create")
  ![CRUD Condition Create](/image/storage/crud-condition/create/a-2.png "CRUD Condition Create")

- Jika sudah terisi klik button Create. Alur  **Create Condition** selesai.
