title: "Edit"

- Buka halaman “Palet -> Condition”, lalu klik button Edit (icon pensil). Akan muncul pop up dengan inputan **Name** untuk memasukkan nama kodndisi, **Slug** untuk membuat name versi uniq (tidak wajib diisi).  
  ![CRUD Edit Condition](/image/storage/crud-condition/edit/a-1.png "CRUD Edit Condition")
  ![CRUD Edit Condition](/image/storage/crud-condition/edit/a-2.png "CRUD Edit Condition")

- Jika sudah terisi klik button **Update.** Alur **Edit Condition** selesai.
