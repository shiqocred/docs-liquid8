- Klik menu â€œMoving Product->Repairâ€ lalu klik button Detail(icon kertas list).
  ![Treatment Repair](/image/storage/repair/treatment/a-1.png "Treatment Repair")
- Terdapat 3 treatment button untuk product repair ini. **Button Edit, Button ToDisplay, Button QCD.**
  ![Treatment Repair](/image/storage/repair/treatment/b-1.png "Treatment Repair")
- **Button Edit** (warna kuning) untuk mengubah data barang jika terjadi pengurangan harga karna rusak, data nama tidak sesuai, dan lain lain. Jika di klik akan muncul pop up untuk meng edit data product. Jika sudah diubah dengan sesuai, klik button Update.
  ![Treatment Repair](/image/storage/repair/treatment/c-1.png "Treatment Repair")
- **Button To Display** (warna biru) untuk memindahkan barang **yang sudah di melalui tahap repairing** ke display. Akan muncul pop up dan klik button confirm jika setuju.
  ![Treatment Repair](/image/storage/repair/treatment/d-1.png "Treatment Repair")
- **Button To QCD** (warna merah) untuk memindahkan barang menuju ke barang QCD, dimana barang ini akan dijual/dibuang secara mandiri. Akan muncul pop up dan jika setuju maka klik confirm. **Ketika barang sudah menuju QCD akan tidak dapat keluar dari QCD. Mohon dengan bijak menggunakan fitur ini.**
  ![Treatment Repair](/image/storage/repair/treatment/e-1.png "Treatment Repair")
- Barang otomatis berpindah dan repair bundle ini tidak akan terhapus (tanda jika sudah selesai maka barang kosong). Alur **Treatment Repair** selesai.
