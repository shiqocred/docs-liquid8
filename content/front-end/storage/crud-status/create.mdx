- Buka halaman “Palet -> Status”, lalu klik button Add Status. Akan muncul pop up dengan inputan **Name** untuk memasukkan nama status, **Slug** untuk membuat name versi uniq (tidak wajib diisi).  
  ![CRUD Status Create](/image/storage/crud-status/create/a-1.png "CRUD Status Create")
  ![CRUD Status Create](/image/storage/crud-status/create/a-2.png "CRUD Status Create")

- Jika sudah terisi klik button Create. Alur **Create Status** selesai.
