title: "Edit"
---

- Buka halaman “Palet -> Status”, lalu klik button Edit Status (icon pensil). Akan muncul pop up dengan inputan **Name** untuk memasukkan nama status, **Slug** untuk membuat name versi uniq (tidak wajib diisi).
  ![CRUD Edit Status](/image/storage/crud-status/edit/a-1.png "CRUD Edit Status")
  ![CRUD Edit Status](/image/storage/crud-status/edit/a-2.png "CRUD Edit Status")

- Jika sudah terisi klik button **Update.** Alur **Edit Status** selesai.
