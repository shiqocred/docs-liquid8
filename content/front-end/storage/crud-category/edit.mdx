---
title: "Edit"
---

- Klik menu “Category Setting -> Sub Category”, lalu klik button Edit (icon pensil). Akan muncul pop up dan isi inputan dengan data yang sudah tersedia. Ubah data tersebut sesuai dengan kehendak. Input **Category Name** untuk meberikan nama kategori, **Discount** untuk memberika potongan persen (%), **Max Price** untuk memberikan batasan barang dengan harga tertinggi.  
  ![CRUD Category Edit](/image/storage/crud-category/edit/a-1.png "CRUD Category Edit")
  ![CRUD Category Edit](/image/storage/crud-category/edit/a-2.png "CRUD Category Edit")
- Jika sudah sesuai, klik button **Update.** Alur **Update Category** selesai.
