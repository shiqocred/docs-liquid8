---
title: "Edit"
---

- Buka halaman “Palet -> List Palet” lalu klik button detail (Paper List) untuk melihat detail palet. Lalu klik button Edit di pojok kanan atas untuk memulai mengedit.
  ![Edit Palet](/image/storage/palet/edit/a-1.png "Edit Palet")
  ![Edit Palet](/image/storage/palet/edit/a-2.png "Edit Palet")

- Lalu tambah atau kurangi product seusia dengan yang dibutuhkan, kilk button Add Produc untuk tambah dan klik button delete (icon tong sampah) untuk mengeluarkan product.
  ![Edit Palet](/image/storage/palet/edit/b-1.png "Edit Palet")

- Jika sudah sesuai semua produk yang diperlukan, selanjutnya mengedit data detail yang dibutuhkan, **Palet Name** untuk memberikan nama palet, **Category** select untuk memberikan kategori palet dan diskon yang akan diberikan, **Warehouse** select untuk memberikan gudang mana yang menampung palet tersebut, **Condition** select untuk memberikan kondisi barang barang yang berada di palet tersebut, **Status** select untuk status barang gagal kirim contohnya, **Description** untuk memberikan deskripsi pada palet, **Brands** pilih “multi select” untuk menammbah banyak product dan pilih “select brand” untuk memilih satu per satu brand.
  ![Edit Palet](/image/storage/palet/edit/c-1.png "Edit Palet")
  ![Edit Palet](/image/storage/palet/edit/c-2.png "Edit Palet")

- Jika daat sudah sesuai, tahap terakhir jika perlu diedit adalah foto palet. Klik button Upload File untuk menambah produk dan klik button x pada pojok kanan atas gambar untuk hapus gambar.
  ![Edit Palet](/image/storage/palet/edit/d-1.png "Edit Palet")

- Scroll ke ats kembali jika proses edit sudah selesai dan klik button Update di pojok kanan atas, alur **Edit Palet** selesai.
