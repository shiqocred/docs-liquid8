title: "Edit"
---

- Buka halaman “Palet -> Brand”, lalu klik button Edit Brand(icon pensil). Akan muncul pop up dengan inputan **Name** untuk memasukkan nama Brand, **Slug** untuk membuat name versi uniq (tidak wajib diisi).
  ![CRUD Edit Brand](/image/storage/crud-brand/edit/a-1.png "CRUD Edit Brand")
  ![CRUD Edit Brand](/image/storage/crud-brand/edit/a-2.png "CRUD Edit Brand")

- Jika sudah terisi klik button **Update.** Alur **Edit Brand** selesai.
