- Buka halaman “Palet -> Brand”, lalu klik button Add Brand. Akan muncul pop up dengan inputan **Name** untuk memasukkan nama Brand, **Slug** untuk membuat name versi uniq (tidak wajib diisi).

![CRUD Brand Create](/image/storage/crud-brand/create/a-2.png "CRUD Brand Create")

- Jika sudah terisi klik button Create. Alur **Create Brand** selesai.
