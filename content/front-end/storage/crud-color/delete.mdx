---
title: "Delete"
description: "Tag color memiliki 2 Type, yaitu untuk WMS dan untuk APK. Jadi Pastikan dahulu tag color mana yang akan di olah."
---

- Klik menu “Category Setting -> Tag Color”, lalu klik button Add Color WMS/APK. Akan muncul pop up dan klik button confirm jika sudah yakin menghapus data. **Penghapusan Data** bersifat **Permanen,** diharapkan penggunaan dengan **Bijak.**  
  ![CRUD Color Delete](/image/storage/crud-color/delete/a-1.png "CRUD Color Delete")
  ![CRUD Color Delete](/image/storage/crud-color/delete/a-2.png "CRUD Color Delete")
- Jika sudah terconfirm maka alur **Delete Tag Color** selesai.
