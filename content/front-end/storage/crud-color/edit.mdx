- Klik menu Category Setting -> Tag Color‚Äù, lalu klik button Add Color WMS/APK. Akan muncul pop up dan isi inputan dengan data yang sudah tersedia. Color Hexa untuk meberikan tag warna berbentuk hexa color, Color Name untuk memberikan tag warna nama yang akan didisplay di list, Fixed Price untuk memberikan harga jika harga lama barang diantara Min Price dan Max Price. Inputan terakhir ialah Min Price dan Max Price berfungsi untuk pembatas range harga barang.
