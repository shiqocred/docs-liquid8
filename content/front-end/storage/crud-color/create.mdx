---
title: "Create"
description: "Tag color memiliki 2 Type, yaitu untuk WMS dan untuk APK. Jadi Pastikan dahulu tag color mana yang akan di olah."
---

- Klik menu “Category Setting -> Tag Color”, lalu klik button Add Color WMS/APK. Akan muncul pop up dan isi inputan yang disediakan. **Color Hexa** untuk meberikan tag warna berbentuk hexa color, **Color Name** untuk memberikan tag warna nama yang akan didisplay di list, **Fixed Price** untuk memberikan harga jika **harga lama** barang diantara **Min Price** dan **Max Price.** Inputan terakhir ialah **Min Price** dan **Max Price** berfungsi untuk pembatas range harga barang.
  ![CRUD Color Create](/image/storage/crud-color/create/a-1.png "CRUD Color Create")
  ![CRUD Color Create](/image/storage/crud-color/create/a-2.png "CRUD Color Create")
- Jika sudah sesuai, klik button **Create.** Alur **Create Tag Color** selesai.
