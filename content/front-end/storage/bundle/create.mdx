title: "Create"
---

- Buka halaman “Moving Product->Bundle”, lalu klik button Add Bundle. Akan diarahkan ke halaman pembuatan bundle.  
  ![ Create Bundle](/image/storage/bundle/create/a-1.png " Create Bundle")
  ![ Create Bundle](/image/storage/bundle/create/a-2.png "Create Bundle")
- Pertama ketika ingin create bundle harus memasukkan Product terlebih. Klik button Add Product lalu akan muncul list product yang bisa kita bundle. Klik button plus pada list tersebut sesuai dengan keinginan. Maka barang akan masuk ke dalam bundle.
  ![ Create Bundle](/image/storage/bundle/create/b-1.png " Create Bundle")
  ![ Create Bundle](/image/storage/bundle/create/b-2.png "Create Bundle")
  ![ Create Bundle](/image/storage/bundle/create/b-3.png "Create Bundle")
- Jika dirasa barang yang ingin dibundle sudah selesai, maka dilanjutkan mengisi detail data pada bundle nya. dengan mengisi **Bundle Name** untuk memberikan nama bundle, **Category** untuk memberikan category dan memotong harga lama dengan diskon yang sudah disetting.
  ![ Create Bundle](/image/storage/bundle/create/c-1.png "Create Bundle")
- Jika dirasa sudah sesuai keseluruhan, maka klik button create di pojok kanan atas. Alur **Create Bundle** selesai.
